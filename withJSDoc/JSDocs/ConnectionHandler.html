Class: ConnectionHandler

ConnectionHandler(messagesObserver, channelsObserver)

new ConnectionHandler(messagesObserver, channelsObserver)

ConnectionHandler is the globale model, it contains two sub-models(messagesObserver and channelsObserver). Once it get updated by the controller, it will tell its two sub-models to update the view.
Parameters:
Name Type Description
messagesObserver object Objet model for messages which will update messagesView.
channelsObserver object Objet model for channels which will update messagesView.
