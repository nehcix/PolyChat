Methods

changeActiveChannel(thisEl)

This method will be used to activate another channel (or update the current channel). It will first empty all the messages in screen, and tell the channelView to change the active channel view (changeActiveChannelView). Then it will send a message of type onGetChannel by using the current channel id, the user input in the textarea, the username and the current time. Finally it will remove the badges of unread messages depending on which channel is activated.
Parameters:
Name Type Description
thisEl objet This holds the span (.channelName) of the channel we want to activate.
